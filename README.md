## 📋 Getting Started

### Prerequisites
- Node.js 16+
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/kamaleshskzone-coder/earthlink-ai.git
cd earthlink-ai

# Install dependencies
npm install

# Start development server
npm run dev
```

### Deployment

Push to the `publish` branch to trigger automatic deployment:

```bash
git push origin main:publish
```

## Installation Instructions

### First Installation (1.0.0)

**Linux/macOS:**
```bash
git clone https://github.com/kamaleshskzone-coder/earthlink-ai.git
cd earthlink-ai
chmod +x setup.sh
./setup.sh
npm run dev
```

**Windows:**
```bash
git clone https://github.com/kamaleshskzone-coder/earthlink-ai.git
cd earthlink-ai
setup.bat
npm run dev
```

**Manual:**
```bash
git clone https://github.com/kamaleshskzone-coder/earthlink-ai.git
cd earthlink-ai
npm install
npm run dev
```
